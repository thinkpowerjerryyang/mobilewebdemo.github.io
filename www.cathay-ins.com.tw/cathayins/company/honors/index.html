連續 5 年
消費者心目中理想品牌大調查『產險業第一名』

2009至2013 年連續五年獲管理雜誌評選為『消費者心目中理想品牌大調查』產險業第一名。
當地客戶優質的保險服務，未來將持續朝「亞太地區最佳金融機構」的目標邁進。

獲獎稱霸台灣

2019

保險卓越獎

國泰產險於第8屆台灣保險卓越獎榮獲「保戶服務」、「資訊應用」和「公益關懷專案企畫」3項金質獎，以及「人才培訓」、「風險管理」和「新興風險保障商品」3項銀質獎。

2019

保險信望愛獎

國泰產險於第21屆保險信望愛獎榮獲「最佳專案業顧問」、「最佳社會貢獻」、「最佳保險專業」、「最佳商品創意」、「最佳社會責任」和「最佳通訊處」6項特優獎，以及「最佳整合傳播」和「最佳保險成就」2項優選。

2019

CSR企業社會責任獎

國泰產險於以「風險停看聽，守護你我他」國泰安全守護計畫，獲得第15屆CSR企業社會責任獎-教育推廣組楷模獎。

2019

金峰獎

國泰產險於第18屆金峰獎榮獲「十大傑出企業」和「十大傑出商品」兩項獎項。

2019

台灣企業永續獎

國泰產險榮獲由台灣永續能源研究基金會舉辦之第12傑台灣企業永續獎「創新成長獎」和「社會共榮獎」。

2019

CSR影響力獎

國泰產險以【父母的牽掛】影片獲得由資誠與坎城國際創意節台灣官方代表共同舉辦「第三屆CSR影響力獎」競賽商業貢獻影響力獎。

2019

信譽品牌獎

國泰產險於第21屆信譽品牌獎榮獲產險公司金獎。

2018

台灣企業永續獎

領先同業推出綠色交通相關保險，積極成為業界先行者，為社會低碳轉型貢獻心力，更以保戶角度出發，提供班機延誤快速理賠創新服務，國泰產險獲頒「第十一屆台灣企業永續獎」-創新成長獎。

2018

CSR 影響力獎

國泰產險製作不意外騎士動畫影片，並遠赴蘭嶼拍攝，透過VR遊戲互動教導蘭嶼孩童學習騎車知識。獲選CSR影響力獎「特優獎」之殊榮。

2018

信譽品牌獎

國泰產險於「第二十屆《讀者文摘》信譽品牌獎」榮獲「產險公司類」金獎。

2017

金峰獎

國泰產險於「第十八屆金峰獎」榮獲「十大傑出企業」和「十大傑出商品」2個獎項。

2017

保險卓越獎

國泰產險於「第七屆保險卓越獎」榮獲「人才培訓卓越獎」和「公益關懷專案企劃卓越獎」2個金質獎，以及「保戶服務卓越獎」、 「資訊應用卓越獎」、 「風險管理卓越獎和「商品創新專案企畫卓越獎」4個銀質獎。

2017

保險信望愛獎

國泰產險於「第十九屆保險信望愛獎」榮獲「最佳專業顧問獎」、 「最佳保險成就獎」、 「最佳通訊處獎」及「最佳整合傳播獎」4個獎項，以及「最佳社會責任獎」和「最佳保險專業獎」兩項優選 。

2017

好險 Action 微電影徵選

國泰產險於「第二屆好險，Action! 微電影徵選活動」榮獲汽機車第三人責任保險-保險業者組「佳作」。

2016

好險 Action 微電影徵選

國泰產險於「第一屆好險，Action! 微電影徵選活動」榮獲「公共意外責任險保險-保險業者組」銅賞獎以及「最佳創意人氣獎」兩項大獎。

2016

運動企業認證

國泰產險受體育署肯定，與國泰金控、國泰人壽及國泰世華銀行成為首屆獲得「運動企業認證」之金融業。

2016

金炬獎

國泰產險於「第十三屆金炬獎」榮獲「年度十大績優企業」及「年度十大績優商品」2個獎項。

2016

國家品牌玉山獎

國泰產險於「第十三屆國家品牌玉山獎」榮獲「傑出企業類」及「最佳產品類」2個獎項，並同時雙雙獲得全國首獎。

2015

保險卓越獎

國泰產險於「第六屆保險卓越獎」榮獲「保戶服務卓越獎」金質獎，以及「人才培訓卓越獎」、「資訊應用卓越獎」、「商品創新卓越獎」、「風險管理卓越獎」4 個銀質獎。

2015

金炬獎

國泰產險於由中華民國傑出企業管理人協會主辦的「第十二屆金炬獎」榮獲「年度十大企業」和「年度十大商品」兩個獎項。

2015

保險信望愛獎

國泰產險於「第十七屆保險信望愛獎」榮獲「最佳社會責任獎」、「最佳商品創意獎」、「最佳保險成就獎」、「最佳專業顧問獎」共4項大獎，以及「最佳保險專業獎」、「最佳通路策略獎」、「最佳通訊處獎-桃園分公司」3項優選。

2014

金峰獎

國泰產險於由中華民國傑出企業管理人協會主辦的「第十六屆金峰獎」榮獲「十大傑出企業」和「十大傑出商品」兩個獎項。

2014

保險信望愛獎

國家品牌玉山獎
