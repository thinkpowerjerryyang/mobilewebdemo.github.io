立即投保
  1. 本網站完成投保旅遊險（不限國內外），就抽2萬元華航機票（市價20,000元華航機票抵用券，抽1名）。
  2. 於本網站完成投保車險（需同時投保強制險+其他任意車險，例如車體險、第三人責任險、或駕傷險等），就抽iPhone 11（市價30,400元，共抽2名）。
  3. 於本網站完成投保住宅火險（含住家保戶傘、租客險等），選擇電子保單（或填寫投保滿意度問卷）就抽Philips氣炸鍋（市價12,900元，共抽1名）。
  4. 於本網站完成投保寵物險，就抽人氣LINE貼圖【柴語錄】超萌周邊商品：20吋行李箱（市價2,980元，抽1名）、無線充電盤（市價1,080元，共抽10名）、行動電源（市價890元，共抽14名）。
  5. 【萬元機票任你選】獎項實際為：華航任選自營航班／航點-經濟艙來回機票２萬元兌換券，並將以掛號寄出。得獎者可於抵用券有效期間內（2020/10/31止），持抵用券正本向華旅網際旅行社兌換（限兌換2020/12/31前出發之航班，以實際訂位艙等為主），並完成各項報名手續及作業。逾期視同放棄使用權益，恕無法展延使用期限，亦不得向主辦單位或華旅網際旅行社請求其他權利或補償。本券可抵用多張機票（不含稅金），超額需由中獎者自行負擔，如兌換金額低於2萬元，恕不退還差額。本券不可變更航點、轉讓或變換現金，使用規範及限制依華旅網際旅行社及航空公司之規定。
  6. 住火險投保滿意度問卷，限活動期間內投保住宅火險之客戶可填寫，若投保完成當下未即時填寫，若投保完成當下未即時填寫，請點選填寫問卷補填寫，以確保抽獎資格。
  7. 活動期間每筆交易成功之保單號碼皆享有一次抽獎機會。例：若持有2筆旅遊險保單+1筆住火險電子保單，可同時參加華航萬元機票（2筆機會）和氣炸鍋（1筆機會）抽獎。

車險優惠總整理

  1. 【強制險優惠】機車強制險最高省240元；汽車強制險優惠250元外，再送全家100元禮券。
  2. 【投保車險加碼抽】活動期間完成車險投保（需同時投保強制險+其他任意車險，例如同一張訂單中保了【強制險+車體險】即符合iPhone11抽獎資格。任意車險意指車體險、第三人責任險、駕傷險等）。
  3. 【早鳥投保專屬好禮】用戶於活動期間，於強制險到期35天前（不含到期當天）在本網站完成車險投保（需同時投保強制險+其他任意車險，例如車體險、第三人責任險、或駕傷險等）即可再抽AirPods耳機。 （例：2020/2/28強制險到期者，2020/1/1~1/24期間投保即符合資格）
  4. 獎項說明：AirPods耳機（市價5,290元，共抽5名）、iPhone11（市價30,400元，共抽2名）、全家禮券（市價100元，不限數量，投保汽車強制險即送）
  5. 活動期間每筆交易成功之保單號碼即有1次抽獎機會，若您符合早鳥資格可再多1次一次抽獎機會。例：若您於1/1投保汽車險（強制險+任意險），即可獲得全家禮券100元，並擁有iPhone 11抽獎機會，若您同時符合早鳥資格，可再抽AirPods耳機，即為越早投保越多優惠喔！

注意事項

  1. 【車險早鳥抽】注意事項：用戶在活動期間，於強制險到期35天前（不含到期當天）在本網站完成車險投保（需同時投保強制險+其他任意險，例如車體險、第三人責任險、或強制險附加駕駛人傷害險等）即可參加抽獎。
  2. 強制險到期前2個月即可投保（例：2020年1月份到期者，2019/11/1起即可投保；2020年2月份到期者，2019/12/1起即可投保；2020年3月份到期者，2020/1//1起即可投保依此類推），於活動期間（2020/1/1-6/30）投保車險者，需同時投保強制+其他任意車險（包含車體險、第三人責任險、駕傷險等），方符合iPhone11抽獎資格。
