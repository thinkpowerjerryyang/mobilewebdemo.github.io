2020/04/08

國泰產險致力公益活動，成為我國產險業界首家獲國際SROI認證企業

不少企業與民眾在從事公益活動時，會思索自身的投入，是否能對社會產生確切的幫助，並好奇其投入是否有產生相應的價值。國泰產險也不例外，於2018年起領先業界，響應國際趨勢，成為我國產險業界首例引入「社會投資報酬」(Social Return on Investment, SROI )的企業，其成效卓越，獲國際社會價值協會（Social Value International, SVI）頒發認證。

 

「不意外騎士專案」資源數倍化 每1元投入就能產生3.30元社會價值

國泰產險長年投入各議題公益活動，自2015年起舉辦「不意外騎士專案」，積極赴各級院校宣導交通、騎行之安全議題，並藉由全方位的媒體宣導，盼將正確的交通觀念傳遞給學子們。而為使資源投放更加精準解決社會問題，國泰產險與資誠聯合會計師事務所（PwC）合作，自2018年起透過SROI評估「不意外騎士專案」公益關懷成效，經國際社會價值協會（Social Value International, SVI）認證符合SROI準則。「不意外騎士SROI評估型報告」(Accident-Free Riders Social Return on Investment Evaluation Report)報告指出，國泰產險「不意外騎士專案」每投入新台幣1元，可產生新台幣3.30元的社會價值，對於利害關係人亦產生不同程度的影響與改變，涵蓋族群包含國泰產險自身、學校教官、學生、合作夥伴等。

 

SROI數據客觀呈現，「不意外騎士專案」影響範圍深入社會各個角落

不僅如此，透過量化數據呈現公益關懷價值，顯現出國泰產險該專案對於經濟面、環境面及社會面，長期所造成的正向衝擊與效益，報告也發現「不意外騎士專案」對於青年「行」的安全具體成效，包含68％學生表示，對於騎車上路更加有信心；80％學生更加認識「交通風險與危險預防」概念；65％學生提升了「危險感知能力與防禦性駕駛技術」，同時，根據追蹤，自2018年起參與不意外騎士活動的學生，皆未有事故發生，連帶影響使學校事故率趨緩。

國泰產險資深副總經理許嘉元表示，產險的核心職能包含「損害防阻」，如何協助客戶、社會大眾減低損害發生的機率，一直是國泰產險致力的課題，也因此長年投入如「不意外騎士」、關注校園安全議題的「不意外學園」、針對專業駕駛員所執行的「駕駛風險評估報告」，以及各式新興議題的保險商品開發。盼能藉由專業能力的發揮，貼合民眾所需，成為最懂客戶需求的保險公司。
